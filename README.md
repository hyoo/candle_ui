# CANDLE UI

This is a User Interface prototype for ECP CANDLE project.



## Getting Started

### Prerequisites
- [yarn package manager](https://yarnpkg.com/en/)
- [elm](http://elm-lang.org/)
```
# for macOS
$ brew install elm yarn
```

### Install
```
$ yarn // need only once
```

### Run
```
$ npm start
```
